# radix

This is a console app that can convert numbers between binary, decimal and hexadecimal.

## Requirements

### Rust Toolchain

```
pacman -S rustup
```

## Installation

Clone the repository:

```
git clone https://github.com/moonflower9/radix.git
cd radix
```

Then build and run:

```
cargo build
cargo run <input_base> <input> <output_base>
```
